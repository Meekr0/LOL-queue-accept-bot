# lol_queueBot

INSTRUCTIONS
1. START 'LOL_QueueAcceptBot.jar
2. Follow the instructions displayed on the screen. These are as follows:
  #1 Move your cursor to the upper-left corner of your League of Legends Client and don't move it
  #2 Wait for the confirmation message to appear on the screen (5 seconds after the black box appears)
  #3 You can move your cursor now. Do not move the black box.
  #4 When a match is found, the program will accept it automatically. It will also automatically reset itself if a match is dodged.
  #5 If you want to do something while waiting for the game to be found, close the black box.
  #6 Even when the program accepts the match, make sure to close the black box afterwards.
